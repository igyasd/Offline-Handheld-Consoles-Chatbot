# 🎮 Offline Handheld Chatbot

A lightweight AI chatbot built for handheld gaming devices and portable systems. Chat with an AI assistant completely offline, without requiring an internet connection, cloud services, or user accounts.

## ✨ Features

* 🤖 Offline AI conversations
* ⚡ Fast local responses
* 🔒 Privacy-focused design
* 💾 Local chat storage
* 📱 Optimized for handheld devices
* 🚫 No internet connection required
* 🛡️ No accounts or subscriptions

## 🎮 Supported Devices

* Nintendo Switch (Homebrew)
* Steam Deck
* ASUS ROG Ally
* Lenovo Legion Go
* Windows handheld PCs
* Linux-based handheld devices

## 📥 Installation

1. Download the latest release.
2. Extract the files to your device.
3. Launch the application.
4. Start chatting.

## 🚀 Usage

1. Open the application.
2. Wait for the AI model to load.
3. Enter a message.
4. Receive an AI-generated response.
